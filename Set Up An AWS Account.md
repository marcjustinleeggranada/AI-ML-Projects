## Introducing Today's Project!

In this project, I'm going to set up my AWS account, including the billing and enabling free tier alerts

### Key tools and concepts

The key tools I used was the AWS Management Console,  Key concepts I learnt include how to manage an AWS Account, recognizing the importance of root user credentials, setting up billing alerts along with multi-factor authentication and understanding the services and usage limits of AWS Free Tier

### Challenges and wins

This project took me approximately 20 minutes The most challenging part was navigating between different sections.

---

## Creating My AWS Account

In this step, I'm going to Sign up for a new AWS account by verifying my email address and creating a strong user password

### Understanding the root user

I learned that a root user is the main account with full access to everything in the AWS account, a strong password is important because if your password is compromised it would allow the unauthorized user to basically have a master key to your account.

---

## Setting Up Billing and Payment

In this step, I'm going to set up my billing by choosing my account plan along with some other required information.

### Why AWS requires a payment method

I learned that AWS requires a payment method because it needs you to prove that you are a real human before using their services.

## Completing Account Activation

In this step, I'm going to verify my identity and select a support plan to activate my account.

### Identity verification and support plan

I verified my identity by using my Gmail account and verifying it through its website. The Basic support plan includes access to AWS documentation, forums and AWS Trusted Advisor checks.

## Enabling Free Tier Alerts

In this step, I'm going to enable my free tier alerts in order to not be charged for my NextWork projects. And as an extra safety net, I'm going to set up my billing alerts so that AWS emails me if i ever get close to a Free Tier limit.

![Image](http://nextwork.ai/mischievous_gray_loyal_tuke/uploads/aws-account-setup_j6k1l7m3)

### Why usage alerts matter

I set up usage alerts because once my usage reaches 85% of the free trial limit, it gives me time to stop or adjust my usage before any charges apply
